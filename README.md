# skills-playbooks

A public home for the practical AI skills and playbooks I use in real workflows.

## Why this exists

Most AI advice is generic. This repo is for the opposite: repeatable, field-tested workflows that are actually useful.

I use this to share **how** I run agent systems in practice — without leaking personal or sensitive details.

## What will live here

- **Skills**
  - Reusable capability modules (clear scope, inputs, outputs, failure modes)
  - Prompt patterns and routing strategies
  - Reliability and guardrail patterns

- **Playbooks**
  - End-to-end workflow recipes for common tasks
  - Operator checklists (before/during/after)
  - Recovery and fallback steps when runs fail

## Redaction policy

Everything here is sanitized for public sharing.

- No private user data
- No secrets
- No internal-only environment specifics
