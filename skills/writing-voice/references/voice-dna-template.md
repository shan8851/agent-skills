# Voice DNA (Template)

Replace placeholders with your own real writing guidance.

## Writing rules
- Write like a sharp human, not a language model.
- Use contractions naturally.
- Keep paragraphs short.
- Get to the point quickly.
- Be specific when making claims.
- Vary sentence length.
- Avoid mechanical transitions.
- Prefer clarity over performative polish.

## Formatting rules
- Paragraphs: 1-3 sentences.
- Prefer digits for numbers.
- Define punctuation preferences (for example: no em dashes).
- Define whether bold is allowed and how often.

## Banned phrases
List phrases that should never appear.

Examples:
- “It’s important to note that...”
- “In today’s landscape...”
- “Leverage”, “utilize”, “unlock”, “supercharge”

## Style adjustments
- Define short-form vs long-form defaults.
- Define tone (for example: direct, conversational, practical, slightly opinionated).
- Define how much humor is acceptable.

## Writing samples
Add 3-6 representative samples from the target writer.

For each sample include:
- what style this sample represents
- short excerpt
- what to mimic (rhythm, vocabulary, structure)

## Review checklist
Before returning output:
- matches tone and pacing
- avoids banned phrases
- follows formatting rules
- preserves intent and CTA
- list-heavy sections converted to prose when better
- unnecessary emoji removed
- em dashes removed if disallowed by voice
- headings normalized away from generic title case when needed
