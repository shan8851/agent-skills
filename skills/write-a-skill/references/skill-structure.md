# Skill Structure

Use this as the default structure:

```
skill-name/
├── SKILL.md
├── references/
│   ├── topic-a.md
│   └── topic-b.md
├── scripts/
│   └── helper.py
└── assets/
    └── template.ext
```

Notes:
- `SKILL.md` is required.
- `references/` is preferred for detailed material.
- `scripts/` is optional.
- `assets/` is optional.
