# Humanizer Mini Pass

Use this as a lightweight cleanup pass after drafting in voice.

## Purpose
Remove obvious AI-style residue without stripping personality.

## Keep this pass small
Only enforce these checks by default:
1. Lists to prose where appropriate
2. Emoji cleanup
3. Em dash cleanup
4. Title-case heading cleanup when it clashes with the writer's voice
5. Cut obvious preamble or recap sentences that only announce the point

## Rules
- Do not rewrite for generic neutrality.
- Do not remove sharp opinions, self-awareness, or natural rhythm.
- Preserve phrasing if it already sounds like the writer.
- If a list is clearer as a list, keep it.
- If a sentence has personality and precision, leave it alone.

## Quick QA
- Still sounds like the writer in the voice samples.
- Reads naturally out loud.
- No obvious AI formatting residue.
- Nothing got sanded down into bland "professional" copy.
