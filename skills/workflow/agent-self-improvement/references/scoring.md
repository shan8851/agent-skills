# Pragmatic Scoring

Use H/M/L only.

- **Impact**: expected value if adopted
- **Recurrence**: how often this issue/opportunity repeats
- **Effort**: implementation effort (H = expensive, L = cheap)
- **Confidence**: quality of evidence supporting recommendation

Guidance:
- Prioritize high impact + high recurrence + low/medium effort.
- If confidence is low, include a short validation step before full adoption.
