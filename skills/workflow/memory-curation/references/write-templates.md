# Write Templates

## Durable memory bullet template
- **<topic> (<YYYY-MM-DD>):** <durable rule/preference/decision in one clear sentence>.

## Daily memory bullet template
- HH:MM — <specific update/decision/context delta>.

## Style
- Be concrete and behavior-oriented.
- Avoid multi-line prose when one line works.
- Include date context when useful for future interpretation.
